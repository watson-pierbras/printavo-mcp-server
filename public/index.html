Printavo MCP Server READ-ONLY

Model Context Protocol server for Printavo print shop management. Connect this to Perplexity as a custom remote connector to query your Printavo data.

search_invoices
Search invoices by date, status, payment, keyword
get_invoice_detail
Full invoice with line items, sizes, pricing
search_customers
Search and browse customers
get_customer_detail
Customer details and contact info
list_statuses
All order status definitions
get_order_stats
Aggregated stats for a date range
get_production_schedule
Orders in production pipeline
get_account_info
Account details and configuration
MCP Endpoint: __PORT_5000__/mcp

Authenticated via Bearer token. No write operations are exposed.
